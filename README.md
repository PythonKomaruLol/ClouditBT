## 🇺🇸 English Version

**Cloudit Mesh** is a decentralized, local messenger concept designed for secure communication without the internet, using **Web Bluetooth API**. It turns your browser into a communication node.

### 🚀 Features
* **Zero-Server Architecture:** No central server, no databases, just peer-to-peer (P2P) signals.
* **Bluetooth Low Energy (BLE):** Uses GATT services to discover and link nearby devices.
* **Glassmorphism UI:** Modern design with "SnowOS" style gradients and blur effects.
* **Terminal Logs:** Integrated debug console to monitor Bluetooth stack activity in real-time.

### 🛠 How to run
1.  Host this repository on **GitHub Pages** (Web Bluetooth requires HTTPS).
2.  Open the link in **Microsoft Edge** or **Google Chrome** (PC or Android).
3.  Ensure your Bluetooth adapter is turned on.
4.  Enter your username and click "Search" to find nearby nodes.

### ⚠️ Technical Requirements
* Browser: Chromium-based (Edge 79+, Chrome 70+).
* Hardware: Bluetooth 4.0+ adapter.
* Context: Must be served over **HTTPS**.

## 🇷🇺 Русская Версия

**Cloudit Mesh** — это концепт децентрализованного локального мессенджера для безопасного общения без интернета, использующий **Web Bluetooth API**. Он превращает твой браузер в узел связи.

### 🚀 Особенности
* **Бессерверная архитектура:** Никаких центральных серверов и баз данных — только прямая связь (P2P).
* **Bluetooth Low Energy (BLE):** Использует GATT-сервисы для поиска и сопряжения устройств поблизости.
* **Glassmorphism UI:** Современный дизайн с градиентами в стиле "SnowOS" и эффектами матового стекла.
* **Терминал-логи:** Встроенная консоль отладки для мониторинга работы Bluetooth-стека в реальном времени.

### 🛠 Как запустить
1.  Залей этот репозиторий на **GitHub Pages** (Bluetooth работает только через HTTPS).
2.  Открой ссылку в **Microsoft Edge** или **Google Chrome** (на ПК или Android).
3.  Убедись, что твой Bluetooth-адаптер включен.
4.  Введи никнейм и нажми «ПОИСК», чтобы найти узлы Cloudit поблизости.

### ⚠️ Технические требования
* Браузер: На базе Chromium (Edge 79+, Chrome 70+).
* Железо: Адаптер Bluetooth 4.0 и выше.
* Окружение: Обязательно наличие **HTTPS** (безопасное соединение).
